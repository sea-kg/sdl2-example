# yourCityIsInvadedByAliens_Tomsk

Game: Your City Is Invaded By Aliens - Tomsk

## Build

Ubuntu requirements 

```
$ sudo apt install \
    cmake g++ \
    libsdl2-dev libsdl2-image-dev
```

MacOS 

```
% brew install sdl2
% brew install sdl2_image 
```
